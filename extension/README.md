# Aletheia Editor Extensions

This directory contains editor integrations for Aletheia.

## Available Extensions

- **vscode/**: VS Code extension
- **shared/**: Shared code and schemas

## VS Code Extension

The primary integration providing:
- Document parsing commands
- Visual preview with overlays
- Insert parsed content to editor
- Virtual document support for Copilot

See [vscode/README.md](vscode/README.md) for details.
